# Interactive Dashboard — Real Estate Market Analysis

The dashboard was developed in Yandex DataLens to visualize key analytical findings.

## Key Features Implemented

### 1. Settlement Type Filter
A filter by settlement type was added to allow users to:
- Focus only on cities
- Compare different types of settlements
- Customize analysis dynamically

---

### 2. Average Area Metric
A new metric **"Average Area"** was added to show:
- Typical apartment size
- Demand patterns by property size

---

### 3. Average Days on Market

The metric **"Average Days on Market"** was added and visualized using an area chart.

This metric:
- Shows average listing duration
- Reflects demand and supply balance
- Is displayed symmetrically with:
  - Number of Listings
  - Average Price

---

### 4. Top-5 Fastest-Selling Locations

A chart was created to display:

**Top 5 settlements with the lowest average days_exposition**

Features:
- Based on average days_exposition
- Sorted ascending
- Color highlighting to emphasize faster sales

This allows quick identification of high-demand areas.

---

### 5. Time Granularity Filter

A time filter was added to allow aggregation by:
- Day
- Week
- Month
- Year

This enables detailed seasonal and trend analysis.

---

## Business Value

The dashboard enables:
- Identification of high-demand locations
- Monitoring of seasonal trends
- Evaluation of property size demand
- Faster strategic decision-making
